- **项目简述**：一个用于青少年心理状态初步评估的移动端Web应用。
- **我的职责**：负责**用户数据看板模块**的**UI还原**与**前端交互开发**。
- **技术实现**：
    1.  使用 **Vue.js + Element-UI** 快速搭建管理后台页面结构。
    2.  利用 **ECharts** 将测评数据（如焦虑指数、压力值）可视化为折线图与柱状图，提升数据可读性。
    3.  采用 **Flex + rem适配方案**，确保数据看板在手机和PC端均有良好显示效果。
- **价值体现**：将抽象的心理数据转化为直观的视觉图表，帮助用户更快理解自身状态。
